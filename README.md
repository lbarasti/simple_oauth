# simple_oauth

A minimal implementation of OAuth Consumer, to be relied upon when implementing OAuth1.0 flows.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     simple_oauth:
       github: lbarasti/simple_oauth
   ```

2. Run `shards install`
